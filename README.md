# Family Asset Tracker

Next.js 16 + Firebase app for tracking family assets, loans, and cross-currency balances.

## Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Firebase Auth, Firestore, and Storage
- pnpm
- Biome 2.4.15

## Local Setup

1. Install Node.js `24.x`
2. Install dependencies with `pnpm install`
3. Copy `.env.example` values into `.env.local`
4. Run `pnpm dev`

## Scripts

- `pnpm dev`
- `pnpm build`
- `pnpm start`
- `pnpm lint`
- `pnpm lint:fix`
- `pnpm format`

## Current Status

Phase 1 scaffolding is in progress. Firebase credentials and production services still need to be connected before the app is fully usable.
